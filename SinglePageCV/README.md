## Cómo ejecutar este proyecto

1. Clona este repositorio o descarga los archivos.  
2. Abre el archivo `index.html` en tu navegador web.  
3. Asegúrate de que el archivo `styles.css` esté en la carpeta correcta y enlazado en el HTML.  
4. Visualiza el proyecto en línea: [SinglePageCV](https://github.com/DeathMariachi/Roadmap.sh-Projects/tree/main/SinglePageCV)
